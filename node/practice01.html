Node.jsの基本・使い方

Node.jsとは

JavaScriptは、ブラウザで動く。それに対しNode.jsはサーバーにもいくことができる。
今まではサーバーとブラウザで違う言語を使うしかなかったが、Node.jsであれば全部一緒にできる。
Node.js で動く便利なパッケージを集めている npm があり、その中に gulp などがある。

nodeのコアモジュールのドキュメントは => https://nodejs.org/api/
コアモジュールを選択すると、その中のプロパティやメソッドが出てくる。
例：File System > fs.appendFile(path, data[, options], callback)

使い方

初期化すると、Package.jsonが作成される。
npmからパッケージをインストールするなら必要なファイル。

      $ npm init

gruntをインストール。—save => Package.jsonに自動的に書かれる。

      $ npm install grunt —save;

node側のコンソール。ブラウザではなく、サーバー側なので、windowはない

      $ node

ファイルを動かす(拡張子いらない)

      $ node app

ファイルを自動で再起動する、nodemonをインストール

      $ sudo npm install nodemon -g

nodemonを実行

      $ nodemon app.js
