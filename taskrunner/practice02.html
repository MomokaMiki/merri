gulpの基本・使い方

gulpを使えるようにする手順

  1. ① Node.jsをインストールする

    Node.jsインストールページ

  2. ② package.json,package-lock.jsonファイルを作成して、パッケージを使えるようにする。

                [ ターミナル ]

            $ npm init

  3. ③ gulpをインストールする

                [ ターミナル ]

            $ sudo npm install -g gulp

gulpのタスクを作成していく

  • ☆ gulpのパッケージをインストール

    次の回(gulpを使ってタスクランナーを作成する )のgulpfile.jsで、変数として保存していっている部分がパッケージになります。

                [ ターミナル ]

            $ npm install [パッケージ名] -—save-dev

  • ☆ gulp バージョン3=>4での注意点

    • ！ 実行の方法を決めなければならない。

      * gulp.series(...tasks) ... 順番に実行する
      * gulp.parallel(...tasks) ... 並列に実行をする

                      [ gulpfile.js ]

                gulp.task('watch', function (done) {
                  gulp.watch("./**/*.html", gulp.series('bs-reload'));
                done();
                });

    • ！ 関数を実行する命令を明記しなければならない。

                      [ gulpfile.js ]

                gulp.task('sass',function(done){
                  // ここにタスクをかく
                  done();
                })
