Vue.js データの登録・更新

データの更新

incrementメソッドでカウントを増やす処理をする。それをv-on属性で指定する。

{{ count }}回クリックしたよ

    [ HTML ]
    <div id="app-1">
      <p>{{ count }}回クリックしたよ</p>
      <button v-on:click="increment">カウント増やす</button>
    </div>
    ーーーーー
    [ JS ]
    var app1 = new Vue({
      el: '#app-1',
      data:{
        count: 0
      },
      methods: {
        increment: function(){
          this.count += 1
        }
      }
    })

SVGデータバインディング

inputにv-model属性を指定し、radiusを指定。radiusプロパティの値をsvgに入れる 
    [ HTML ]
    <div id="app-2">
      <svg xmlns="http://www.w3.org/2000/svg" version="1.1">
        <circle cx="100" cy="75" v-bind:r="radius" fill="lightpink" />
      </svg>
      <input type="range" min="0" max="100" v-model="radius">
    </div>
    ーーーーー
    [ JS ]
    var app2 = new Vue({
      el: '#app-2',
      data:{
        radius: 50
      }
    })
