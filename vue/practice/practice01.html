Vue.js ディレクティブの理解

基本メッセージの表示

{{ message }}で、dataのmessageの内容取ってくる

{{ message }}

v-bind : ディレクティブが初めて要素と紐づいた時に属性に指定された値を入れる

v-bind属性にtitleを指定。dataのmessageの内容取ってくる

マウスを持ってくるとtitle属性に指定したメッセージを表示します

v-if : 属性値に指定した値を判定して表示/非表示

v-if名をseen、dataのseenをtrueにすると文字表示・falseにすると文字非表示

v-if属性にseenプロパティを指定。seenがtrueなのでこのメッセージが表示されています。

v-for : 繰り返し処理

v-for属性にitem in listを指定。listプロパティの中の配列を繰り返して描画

{{ item }}

v-on : イベント利用

v-on属性のclickにhandleClickを指定。methodsのhandleClickのイベントを実行する。

v-model : フォーム入力同期

v-model属性にmessageを指定。dataのmessageに登録され、それを表示できる。.numberを付与で数値に。

{{ message }}
