Vue.js ディレクティブの理解

基本メッセージの表示

{{ message }}で、dataのmessageの内容取ってくる

{{ message }}

        [ HTML ]

        <div id="app">
            <p>{{ message }}</p>
        </div>

        ーーーーー

        [ JS ]

        var app = new Vue({
            el: '#app',
            data: {
                message: 'メッセージを表示！'
            }
        })

v-bind : ディレクティブが初めて要素と紐づいた時に属性に指定された値を入れる

v-bind属性にtitleを指定。dataのmessageの内容取ってくる

マウスを持ってくるとtitle属性に指定したメッセージを表示します 

        [ HTML ]

        <div id="app-2">
            <p v-bind:title="message">
                マウスを持ってくるとtitle属性に指定したメッセージを表示します
            </p>
        </div>

        ーーーーー

        [ JS ]

        var app2 = new Vue({
            el: '#app-2',
            data: {
                message: 'メッセージを表示！' + new Date().toLocaleString()
            }
        })

v-if : 属性値に指定した値を判定して表示/非表示

v-if名をseen、dataのseenをtrueにすると文字表示・falseにすると文字非表示

v-if属性にseenプロパティを指定。seenがtrueなのでこのメッセージが表示されています。 

        [ HTML ]

        <div id="app-3">
            <p v-if="seen">
                v-if属性にseenプロパティを指定。seenがtrueなのでこのメッセージが表示されています。
            </p>
        </div>

        ーーーーー

        [ JS ]

        var app3 = new Vue({
            el: '#app-3',
            data: {
                seen: true
            }
        })

v-for : 繰り返し処理

v-for属性にitem in listを指定。listプロパティの中の配列を繰り返して描画

{{ item }} 

        [ HTML ]

        <div id="app-4">
            <p v-for="item in list">
                {{ item }}
            </p>
        </div>

        ーーーーー

        [ JS ]
        var app4 = new Vue({
            el: '#app-4',
            data: {
                list: [ 'list1', 'list2', 'list3' ]
            }
        })

v-on : イベント利用

v-on属性のclickにhandleClickを指定。methodsのhandleClickのイベントを実行する。

        [ HTML ]

        <div id="app-5">
            <button v-on:click="handleClick">Click</button>
        </div>

        ーーーーー

        [ JS ]

        var app5 = new Vue({
            el: "#app-5",
            methods: {
                handleClick: function(e){
                  alert(event.target)
                }
            }
        })

v-model : フォーム入力同期

v-model属性にmessageを指定。dataのmessageに登録され、それを表示できる。.numberを付与で数値に。

{{ message }}

        [ HTML ]

        <div id="app-6">
            <p>{{ message }}</p>
            <input v-model="message">
        </div>

        ーーーーー

        [ JS ]

        var app6 = new Vue({
            el: "#app-6",
            data: {
                message: '初期メッセージ'
            }
        })
