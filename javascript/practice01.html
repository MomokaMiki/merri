forEach - 繰り返し処理

forEachの使い方

変数の数分処理がされる。

      var colors = ['red', 'green', 'blue'];

      colors . forEach ( function ( color ) {
        console . log ( color );
      })

変数の数分、関数を実行する

関数名の後ろに()をつけるとエラーになってしまうので注意。

        var numbers = [ 1 , 2 , 3 ];
        var sum = 0;
        
        function adder( number ){
          sum += number;
        }
        
        numbers . forEach( adder );

        ーーーーー
        6

forEachのメリット

deleteEmail関数を使って、emailに指定したものを削除
簡単にできる！

        emails . forEach( function ( email ) {
          deleteEmail ( email );
        });
